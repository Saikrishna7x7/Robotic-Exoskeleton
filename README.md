# Neuropathy Disease Detection for Robotic Exoskeleton (MATLAB)

## 📌 Project Overview
This project focuses on detecting neuropathy disease conditions using a trained machine learning model in MATLAB.  
The prediction output is intended to support adaptive control in a robotic exoskeleton system.

The system includes:
- Data preprocessing
- Model training
- Model evaluation
- Prediction module

## 🎯 Objective
To assist robotic exoskeleton control by identifying neuropathy-related abnormalities using machine learning techniques.

## 🧠 Methodology
1. Data collection and preprocessing
2. Feature extraction
3. Model training in MATLAB
4. Performance evaluation
5. Real-time prediction logic

## 🛠 Tools & Technologies
- MATLAB
- Machine Learning Toolbox
- Data processing scripts

## 📊 Key Features
- Trained classification model
- Prediction function for new input data
- Modular code structure
- Designed for integration with exoskeleton control system

## 🔄 Future Scope
- Convert model implementation to Python
- Integrate with real-time sensor data
- Hardware deployment with embedded system
